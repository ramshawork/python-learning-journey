# Week 01 – Python Basics

This week covers Python fundamentals.

## Mini Projects
Number Guessing Game – The computer selects a random number, and the user guesses.
Simple Calculator – Add, subtract, multiply, and divide.
Even/Odd Checker – User input number → check even or odd.
Temperature Converter – Celsius ↔ Fahrenheit.
Multiplication Table Generator – Input number → print table.
Simple Quiz Game – 3–5 questions, score at the end.
Palindrome Checker – Word or number palindrome check.
Simple Dice Roller – Random dice roll simulation.
Age Calculator – Input birth year → calculate age.
Tip Calculator – Bill input → calculate tip & total.

## Focus
- Writing clean code
- Understanding logic, not memorizing syntax
